## Description

Chatterbox is a real-time messaging platform designed to provide fast, secure, and reliable communication. Built with modern web technologies, it enables users to connect instantly through private conversations and group chats. Whether you're coordinating with a team, chatting with friends, or building a community, Chatterbox delivers a smooth messaging experience with end-to-end encryption and real-time message delivery.

**Key Features:**
- Real-time message synchronization
- Secure, encrypted communications
- Group chat support
- User presence indicators
- Message history and search
- File sharing capabilities
- Cross-platform compatibility

## Installation

To install and run Chatterbox locally, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/GU-Web-Dev-2025/final-project-harrison-sheldon
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:3000`

## Usage

After installation, you can start using Chatterbox:

1. **Create an Account**: Sign up with your email and create a secure password
2. **Start a Conversation**: Click "New Chat" to begin a private conversation or create a group
3. **Send Messages**: Type your message and press Enter or click Send
4. **Share Files**: Click the attachment icon to share images, documents, or other files
5. **Search**: Use the search bar to find specific messages or conversations
