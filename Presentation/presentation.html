ChatterBox

https://chatterbox-9e5d6.web.app

Harrison Sheldon

Tue Dec 6 2024

The project focuses on creating a full-stack web-based chat messaging platform that allows real-time messaging, group chats, photo sharing, persistent message history across sessions, and intuitive navigation to allow users to communicate with their friends.

Functional Requirements:

  1. Users must be able to register a username and password

  2. Users must be able to log in with a registered username and password

  3. Users must be able to add other users as friends using their email address

  4. Users must be able to send private messages to other registered users

  5. Users must be able to view message history that persists between sessions

  6. Users must be able to send pictures to other users

  7. Users must be able to navigate between pages

  8. Users must be able to logout of their account

  9. Users must be able to create group chats with other users

  10. Users must be able to leave and delete a group chat
